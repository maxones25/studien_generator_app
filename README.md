## Getting started

## Development

### Installieren der Datenbank

Starten des lokalen Datenbankservers

```console
docker-compose -f "dev-docker-compose.yaml" up -d
```

### Installieren des Backends

Navigiere in den Order backend

```console
cd backend
```

Starte den lokalen Backendserver

```console
npm run start:dev
```
